                                      oneAPI for AMD GPUs (beta) 2024.1.2
                                      thumb_up arrow_forward_ios

                                      oneAPI for AMD GPUs (beta) is a plugin for Intel® oneAPI Toolkits that enables developers to build oneAPI applications with DPC++ / SYCL™ and run them on AMD GPUs.

                                      Note

                                      Please keep in mind that this is beta-quality software, which means that it contains most of the major features, but is not yet complete and still contains known and unknown bugs. For more details on the supported features, see the Features documentation.

                                      The plugin adds a HIP backend to DPC++ and you will see the terms “oneAPI for AMD GPUs (beta)” and “DPC++ HIP plugin” used interchangeably throughout this documentation.

                                      To learn more about oneAPI, see Intel’s oneAPI Overview.

                                      To get started quickly with oneAPI for AMD GPUs (beta), please view the Install oneAPI for AMD GPUs (beta).
